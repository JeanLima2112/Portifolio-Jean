ALGUNS PROJETOS

Gerenciador de Tarefas

Este projeto é uma aplicação completa de gerenciamento de tarefas, que visa simplificar a organização e o acompanhamento de atividades diárias. Ele é composto por uma robusta API desenvolvida em NestJS, um frontend interativo em React e uma versão mobile que utiliza o Ionic Framework . Para garantir a escalabilidade e a segurança dos dados, utilizamos um banco de dados em nuvem com PostgreSQL, proporcionando um armazenamento eficiente e confiável.

Git Hub Search

Um Buscador de Perfis da plataforma GitHub usando tecnologias web como React , Chakra e TS foi possível criar esse Buscador de Usuários. Usando também Axios para Fazer requisições do tipo GET e possibilitar a integração com a API do GitHub

Pong.Py

Este projeto faz parte de um trabalho desenvolvido por Hobbie tem o intuito de ser uma recriação do clássico jogo Pong utilizando a linguagem de programação Python. Para a construção do jogo, foram implementados conceitos de programação orientada a objetos e design de interfaces gráficas com a biblioteca Pygame. O projeto também incluiu a aplicação de princípios de jogabilidade e interatividade, tornando o jogo não apenas funcional, mas também envolvente e divertido para os usuários.
