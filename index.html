Gerador De Senhas

Um gerador de senhas é uma ferramenta essencial para garantir a segurança online. utilizando tecnologias web como HTML , CSS e JS foi possível criar um gerador de senhas eficiente e visualmente atraente. essa ferramenta pode ser facilmente integrada a qualquer site ou usada como uma aplicação independente para gerar senhas seguras de forma rápida e eficiente. a ferramenta é projetada para ser usada tanto em desktops quando em dispositivos móveis garantindo uma experiência de usuário agradável e consistente em diferentes tamanhos de tela.

Pong.Py

Este projeto faz parte de um trabalho desenvolvido por Hobbie tem o intuito de ser uma recriação do clássico jogo Pong utilizando a linguagem de programação Python. Para a construção do jogo, foram implementados conceitos de programação orientada a objetos e design de interfaces gráficas com a biblioteca Pygame. O projeto também incluiu a aplicação de princípios de jogabilidade e interatividade, tornando o jogo não apenas funcional, mas também envolvente e divertido para os usuários.
